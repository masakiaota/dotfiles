### 概要
Hyper+xonshでお手軽かつモダンな環境づくり

### 準備
- Hyper https://hyper.is
- xonsh https://xon.sh/osx.html

### 今まで何をしたか
- Hyperをインストール
    - hyper i でいくつか拡張機能をインストール
    - textsize変更
    - デフォルトのshellをxonshに設定
- xonsh
```
brew install python3
pip3 install xonsh
pip3 install gnureadline
```
でインストール
```
brew install xonsh
```
でもインストールできるが推奨しない
